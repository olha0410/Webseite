PYTHON

Python ist, genau wie HTML oder Java Script, eine Programmiersprache, die Anfangs ziemlich einschüchternd wirken kann. Aber das lernen der grundlagen kann helfen.
In diesem Workshop geschah genau das. Es war sehr nachvollziehbar, wie Funktionen, Variablen, Klassen oder Print verwendet werden, um alle möglichen Programme zu programmieren. Natürlich gibt es nach den Grundlagen noch viel mehr Dinge. Es wird immer komplizierter, aber alles, was man für den Anfang wirklich braucht, sind die Grundlagen. Während des Workshops konnten wir unseren eigenen Passwort-Generator (oder was auch immer wir wollten) mithilfe von PyCharm planen und programmieren, um die Verwendung einfacher Python-Methoden zu üben. Gegen Ende haben wir zusammen mit dem Referent ein Hangman-Spiel programmiert um es darauf ausprobieren zu können.

PyCharm

PyCharm ist eine Entwicklungsanwendung für die Programmierung von Python. PyCharm bietet einen Debugger, Codeanalyse und Module für Java Script oder CSS

Passwort-Generator

Den Passwort-Generator mussten wir mithilfe den Python-Grundlagen programmieren. Dazu konnte man mit Funktionen, Variablen, Klassen und der Print Funktion arbeiten. Das Program sollte wenn man auf "Run" klickt, dir die Option geben, möglich Passwörter zu generieren. Man soll die Länge auswählen können. Falls Fehler auftreten sollten, kann man mithilfe von dem eingebauten Debugger und Fehlerorter das Program verbessern.

Hangman-Spiel

Dies musste das Program für das Spiel können:
- Option, Buchstaben zu erraten.
- Wenn ein Buchstabe nicht vorkommt im Wort, werden mehr Körperteile des Hangmans hinzfügt.
- Wenn das Wort richtig erraten ist, gewinnt man.
- Wenn der Hangman komplett ist, verliert man.
